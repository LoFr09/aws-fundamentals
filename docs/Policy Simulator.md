# Policy Simulator

## Simulator vs. Live-Test

Ich vermute aus diesen zwei Gründen:
    - Ich lösche beim beim Live-Test echte Sachen, bei der Simulation sind das nur versuche, dadurch geht sicher nichts kapput.
    - Wenn man mehrere Policies gleichzeitig Testen möchte geht das so viel schneller.
    - Beim Live Test könnte man ausversehen was löschen und wüsste dan auch nicht so schnell wieso das so ist.
